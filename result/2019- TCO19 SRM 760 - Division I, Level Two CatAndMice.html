Problem Statement for "CatAndMice"

Problem Statement

We are on an infinite horizontal plane. For each pair of integers (x, y) such that |x| &lt;= N and |y| &lt;= N there is an animal sitting on the point (x, y). The animal on (0, 0) is a cat and all other animals are mice.

The cats wants to run away in some direction, that is, along some infinite ray that starts at (0, 0). Along the way the cat will catch and eat all the mice it encounters.

The cat wants to catch exactly C mice. Return the number of different directions the cat can choose.

Definition

Class:
CatAndMice
Method:
countDirections
Parameters:
int, int
Returns:
long
Method signature:
long countDirections(int N, int C)
(be sure your method is public)

Constraints

  • N will be between 1 and 5,000,000, inclusive.
  • C will be between 1 and N, inclusive.
