Problem Statement for "VennDiagrams"

Problem Statement

You are given an int N. Produce a bitmap drawing of any valid Venn diagram for N sets.

To avoid confusion, we will use the word classes to denote the N sets for which you are constructing the diagram. We will number the classes 0 through N-1.

Your task is as follows:

  1. Choose the dimensions for the bitmap. It must be a rectangle with at most 50 rows and at most 50 columns.
  2. For each class, choose a subset of pixels of the bitmap that will represent the class.
  3. Color the pixels by assigning the color number (2c1 + ... + 2ck) to a pixel that represents the classes { c1, ..., ck }.

Additionally, the drawing must have some properties you would intuitively expect from a Venn diagram. In order to define these properties formally:

  • Imagine that we placed the bitmap onto an infinite grid of pixels of color 0.
  • Once we do that, let's say that a block denotes a set of pixels B on the infinite grid such that all pixels that do belong to B form a single nonempty 4-connected area, and all pixels that do not belong to B also form a single nonempty 4-connected area.

Your drawing must satisfy the following:

  • For each nonempty subset of classes, the pixels that represent all of them (including pixels that also represent some other classes) form a block.
  • For each subset of classes (including the empty set), the pixels that represent exactly all of them (and no other classes) form a block.

If the bitmap you constructed has R rows and C columns, return a int[] with 2+R*C elements: { R, C, color[0][0], ..., color[0][C-1], color[1][0], ..., color[R-1][C-1] }. Any valid solution will be accepted.
