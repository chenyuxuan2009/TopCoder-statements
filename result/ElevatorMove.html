Problem Statement for "ElevatorMove"

Problem Statement

A building is serviced by a single elevator. The elevator services requests to go up or down. If an elevator receives a request while it is not moving, it starts servicing the request immediately. If a request comes while the elevator is in motion, the elevator processes the new request immediately only if doing so does not require the elevator to change its direction. Otherwise, the elevator finishes servicing its current request before servicing the new request. Given the current position of the elevator, a destination to which the elevator is moving, and the source of a new request, calculate the number of floors the elevator needs to travel before it could start servicing the new request.

The position argument represents the elevator's current floor. The destination argument represents the destination floor to which the elevator is moving now. If the position equals the destination, the elevator is not moving.

The request argument represents the floor on which the new request originates. The direction argument indicates whether the request is to go up (+1) or down (-1). For example, an input combination of position=4, destination=10, request=5, and direction=-1 describes a situation when the elevator is passing the fourth floor on its way to the tenth floor, while the request to go down is coming from the fifth floor.

Your method should return the number of floors the elevator needs to travel before it could start servicing the new request. Assume that the elevator has no other requests to service.

Definition

Class:
ElevatorMove
Method:
distance
Parameters:
int, int, int, int
Returns:
int
Method signature:
int distance(int position, int destination, int request, int direction)
(be sure your method is public)

Constraints

  • position, destination, and request are numbers between 1 and 12, inclusive.
  • direction is either 1 indicating a request to go up, or -1 indicating a request to go down.
  • The direction of a request coming from the top floor is always -1.
  • The direction of a request coming from the first floor is always 1.
