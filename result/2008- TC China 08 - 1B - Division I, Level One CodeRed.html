Problem Statement for "CodeRed"

Problem Statement

We are looking for occurrences of the string &quot;red&quot; in a piece of text. For example, in the text 

    the detective questioned his credibility

we find &quot;red&quot; inside the word &quot;credibility&quot;.

Capitalization doesn't matter, so in the text 

    Server ERRED in Redirecting Spam

we find &quot;red&quot; at the end of &quot;ERRED&quot;, as well as at the beginning of &quot;Redirecting&quot;.

Given a String, count the number of times &quot;red&quot; appears in it with any capitalization.

Definition

Class:
CodeRed
Method:
count
Parameters:
String
Returns:
int
Method signature:
int count(String text)
(be sure your method is public)

Constraints

  • text will contain between 1 and 50 characters, inclusive.
  • text will contain only letters ('A'-'Z', 'a'-'z') and spaces (' ').
