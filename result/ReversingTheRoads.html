Problem Statement for "ReversingTheRoads"

Problem Statement

In some country every two distinct cities are connected by one-way road. You are the road architect and you wish that people of the country be able to travel between every possible pair of cities using the roads. To achieve your aim you can change directions of some roads to the opposite.

You will be given String[] cost. j-th character of i-th element of cost will be '-', if there is no road, which goes from city i to city j. Otherwise, j-th character of i-th element of cost will encode cost of reversing the direction of the road, which goes from city i to city j (characters 'a', 'b', ..., 'z' encode numbers 1, 2, ..., 26; characters 'A', 'B', ..., 'Z' encode numbers 27, 28, ..., 52). Return the minimum possible cost, which is needed to achieve your aim.

Constraints

  • cost will contain between 3 and 50 elements, inclusive.
  • The number of characters in each element of cost will be the same as the number of elements in cost.
  • All characters in all elements of cost will be '-', 'a'..'z' or 'A'..'Z'.
  • i-th character in i-th element of cost will be '-'.
  • Exactly one of i-th character in j-th element of cost and j-th character in i-th element of cost will be '-'.
