Problem Statement for "ShippingFee"

Problem Statement

Calculate how much it would cost to ship a package from sourceZipCode to destZipCode. The shipping service uses the following formula to determine shipping cost: weight * rate * (zoneDifference + 1). You are given a String[], zones, which defines the locations of various zip codes. Each element of zones is a single space delimited list of five-digit integers. Element i (zero-indexed) of zones contains all the zip codes located in zone i. zoneDifference is defined as the absolute value of the difference between the zone of sourceZipCode and the zone of destZipCode.

Definition

Class:
ShippingFee
Method:
getFee
Parameters:
int, int, int, int, String[]
Returns:
int
Method signature:
int getFee(int weight, int rate, int sourceZipCode, int destZipCode, String[] zones)
(be sure your method is public)

Constraints

  • weight will be between 1 and 100, inclusive.
  • rate will be between 1 and 10, inclusive.
  • sourceZipCode and destZipCode will each be between 10000 and 99999, inclusive.
  • zones will contain between 1 and 50 elements, inclusive.
  • Each element of zones will contain between 5 and 47 characters, inclusive.
  • Each element of zones will be a single space delimited list of one or more five-digit integers, with no leading or trailing spaces.
  • zones will contain no duplicate integers.
  • Both sourceZipCode and destZipCode will exist in zones.
