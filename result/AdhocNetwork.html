Problem Statement for "AdhocNetwork"

Problem Statement

You have deployed an ad hoc network to take temperature readings in a test area. When data from the network arrives back at the main control device it has the form "temp x y". temp is the temperature reading and x,y are the coordinates of the device sampling the temperature. Given two ints X and Y describing the location to measure, you must find which readings are most relevant. More precisely, given a String[] readings containing data of the form specified earlier, you will return the temperature which is closest to the location X,Y. Closeness is measured by the standard distance formula:
	dist = sqrt( (x-X)^2 + (y-Y)^2 ).
For convenience, no two readings in data will have the same distance from the point X,Y.

Definition

Class:
AdhocNetwork
Method:
bestTemp
Parameters:
String[], int, int
Returns:
int
Method signature:
int bestTemp(String[] data, int X, int Y)
(be sure your method is public)

Constraints

  • data will contain between 2 and 50 elements inclusive.
  • Each element of data will have the form (quotes for clarity) "temp x y" where temp, x, and y are integers between 0 and 100 inclusive with no extra leading zeros.
  • X will be between 0 and 100 inclusive.
  • Y will be between 0 and 100 inclusive.
  • No two readings in data will have the same distance from the point X,Y.
