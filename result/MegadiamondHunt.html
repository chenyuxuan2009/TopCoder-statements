Problem Statement for "MegadiamondHunt"

Problem Statement

NOTE: This problem statement contains superscripts that may not display properly if viewed outside of the applet.

You are a megadiamond hunter looking for diamonds in a peculiar mine. The mine is a String containing only '&lt;' and '&gt;' characters. Each megadiamond is a substring containing one or more consecutive '&lt;' characters immediately followed by the same number of consecutive '&gt;' characters. For example, "&lt;&gt;", "&lt;&lt;&gt;&gt;" and "&lt;&lt;&lt;&lt;&gt;&gt;&gt;&gt;" are all megadiamonds. Each time you find a megadiamond, you can remove it, and the corresponding characters are removed from the mine.

For example, if the mine looks like this:

&gt;&lt;&lt;&gt;&lt;&gt;&gt;&gt;&lt;

You can start by removing the first occurrence of "&lt;&gt;" to get:

&gt;&lt;&lt;&gt;&gt;&gt;&lt;

You can then remove the megadiamond "&lt;&lt;&gt;&gt;" to get:

&gt;&gt;&lt;

There are no more megadiamonds, so your expedition is done.

The size of a megadiamond is the number of '&lt;' characters it contains. The value of a megadiamond of size n is n2. The total income of your expedition is the sum of the values of the megadiamonds you removed from the mine. Given a String mine, return the maximum possible income you can get from an expedition.
