Problem Statement for "ProcessingRequests"

Problem Statement

Write a program to receive requests from two clients, process them, and obtain the summary of requests.

Given class RequestsUtils:

class RequestsUtils {
  /**
   *  Enumeration of clients.
   */
  public enum ClientNumber {
    Client1, Client2;
  };

  /**
   *  Creates a RequestsUtils object with the given data seed.
   * 
   *  @param dataSeed An integer representing the data seed.
   */
  public RequestsUtils(int dataSeed);

  /**
   * Gets the client object for a given client number.
   *
   * @param client A ClientNumber value representing a client.
   * @return The Client object for the given client.
   */
  public Client getClient(ClientNumber client);

  /**
   *  Processes the given request from the given client. This method works for about
   *  50 milliseconds and blocks the current thread until the request is complete.
   *  The method can be called from at most one thread at the same time. 
   * 
   *  @param client The ClientNumber for the request.
   *  @param data The Data to be processed.
   */
  public void processRequest(ClientNumber client, Data data);

  /**
   *  Gets the summary of all requests processed by this object.
   * 
   *  @return The compressed summary of processed requests.
   */
  public int getRequestsSummary();
}

Given class Data:

Represents data that needs to be processed.

class Data {
  /**
   * Gets whether or not the Data is empty.
   * 
   * @return True if and only if the Data is non-empty.
   */
  public boolean isEmpty();
}

Given class Client:

A client will have at most 15 requests to process data.

class Client {
  /**
   *  Blocks until the next request is available. If there are no more requests from this client,
   *  empty data is returned and does not need to be processed. Waiting for a request can
   *  take some time - at most 100 milliseconds. Only one thread may call this method per instance.
   *
   *  @return The Data to be processed.
   */
  public Data waitForRequest();
}

DEFINITION

Define public class ProcessingRequests with public method int getResult(int dataSeed).

getResult must be implemented as follows:

GET CLIENTS

Create RequestsUtils utils by passing dataSeed to RequestsUtils(int).

Use utils.getClient(ClientNumber) to get both Client instances.

PROCESS THE REQUESTS

Process all requests from both clients by using the processRequest() method. The requests can be processed in any order, but requests from the same client must be processed in the order that they are received.

GET SUMMARY OF REQUESTS

After all the requests are processed, return utils.getRequestsSummary().

Notes

  • The execution time of your program must not exceed 2 seconds. You are recommended to use multithreading to decrease the execution time in this problem.

Constraints

  • dataSeed will be between 0 and 1,000,000,000, inclusive.
