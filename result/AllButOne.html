Problem Statement for "AllButOne"

Problem Statement

We call a sequence A an (N,S)-sequence if it contains N positive integers (numbered 0 through N-1) and the sum of its elements is equal to S.


You're given two ints, S and MOD. You're allowed to pick an arbitrary positive integer N and an arbitrary (N,S)-sequence A. Then, you should compute P = (A[0]*A[1]*...*A[N-1]) modulo MOD.


Compute and return the number of different values of P that can be obtained this way.

Definition

Class:
AllButOne
Method:
countRemainders
Parameters:
int, int
Returns:
int
Method signature:
int countRemainders(int S, int MOD)
(be sure your method is public)

Constraints

  • S will be between 1 and 500, inclusive.
  • MOD will be between 2 and 50,000, inclusive.
