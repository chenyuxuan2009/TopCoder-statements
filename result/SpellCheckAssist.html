Problem Statement for "SpellCheckAssist"

Problem Statement

When typing text, people often make small mistakes. Most modern editors provide tools to quickly fix such mistakes with the help of spell checkers. When the user misspells a word, the editor suggests a number of correct words that are similar to what the user typed.

Let us call words A and B similar if A can be transformed to B by exactly one of the following operations:

  • Insert one letter at any position.
  • Remove one letter at any position.
  • Change one letter at any position to any other letter.
  • Swap two adjacent letters.

You are given a String word and a String[] dictionary. word will not be equal to any element of dictionary. Return a int[] containing the 0-based indices of all the words in dictionary that are similar to word. The returned int[] must be sorted in ascending order.

Definition

Class:
SpellCheckAssist
Method:
suggestFixes
Parameters:
String, String[]
Returns:
int[]
Method signature:
int[] suggestFixes(String word, String[] dictionary)
(be sure your method is public)

Constraints

  • word will contain between 1 and 50 characters, inclusive.
  • word will contain only lowercase letters ('a'-'z').
  • dictionary will contain between 1 and 50 elements, inclusive.
  • Each elemenent of dictionary will contain between 1 and 50 characters, inclusive.
  • Each element of dictionary will contain only lowercase letters ('a'-'z').
  • All elements of dictionary will be distinct.
  • word will not be equal to any element of dictionary.
