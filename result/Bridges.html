Problem Statement for "Bridges"

Problem Statement

Bridges is a logic puzzle where the challenge is to connect a number of islands on a grid using horizontal and vertical bridges so the following conditions hold:

  • The number of bridges connected to an island matches the given number for that island
  • At most 2 bridges connect the same two islands
  • Bridges may not cross each other, or other islands
  • It is possible to go from one island to all other islands

The figure above to the left shows the puzzle layout, and the picture above to the right the unique solution. No two islands will lie next to each other, horizontally or vertically. Only puzzles that have one unique solution are valid Bridges puzzles.

Create a class Bridges containing the method solve which takes a String[] puzzle containing the puzzle, and returns a String[] containing the solution. In puzzle, empty grid squares will be represented by a period, '.', and islands by a digit between '1' and '8', inclusive (the exact number of bridges this island should have). The solution should return the same map, except that some of the empty grid squares should be replaced by single or double bridges. The characters '-' and '=' should be used for single and double horizontal bridges, respectively, and the characters '|' and 'H' for single and double vertical bridges, respectively.

Constraints

  • puzzle will contain between 1 and 10 elements, inclusive.
  • Each element in puzzle will contain between 1 and 10 characters, inclusive.
  • Each element in puzzle will contain the same number of characters.
  • puzzle will only contain the character '.' and digits between '1' and '8', inclusive.
  • puzzle will contain at least two islands.
  • No two digits in puzzle will be adjacent to each other, horizontally or vertically.
  • The puzzle will have exactly one solution.
