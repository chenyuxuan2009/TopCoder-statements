Problem Statement for "PalinBox"

Problem Statement

You will be given a rectangular box of letters as a String[] (each element is a row). We want to change box so that every column and every row is a palindrome. That is, reading a row left-to-right is the same as right-to-left and reading a column top-to-bottom is the same as bottom-to-top. Return the fewest number of cells in box that need to be changed to achieve this effect.

Definition

Class:
PalinBox
Method:
getMin
Parameters:
String[]
Returns:
int
Method signature:
int getMin(String[] box)
(be sure your method is public)

Notes

  • box will contain between 2 and 50 elements, inclusive.
  • Each element of box will contain between 2 and 50 characters, inclusive.
  • Each character in box will be a capital letter ('A'-'Z').
  • Each element of box will contain the same number of characters.

Constraints

    3. { "ABC", "DEF", "GHI" }
