Problem Statement for "MakingRightyTriangle"

Problem Statement

There is a 2-D plane and there are N points. i-th point is located at (X[i], Y[i]). You are going to choose 3 distinct points from them. Then, we calculate: 
longest = the longest distance between two of the three points
shortest = the shortest distance between two of the three points
You want to pick the three points that minimize the value, longest/shortest. Given int[] X and int[] Y, return this smallest value.

Definition

Class:
MakingRightyTriangle
Method:
best
Parameters:
int[], int[]
Returns:
double
Method signature:
double best(int[] X, int[] Y)
(be sure your method is public)

Constraints

  • The number of elements in X will be between 3 and 200, inclusive.
  • The number of elements in Y will be between 3 and 200, inclusive.
  • X and Y will have the same number of elements.
  • Each element in X will be between -1,000 and 1,000 inclusive.
  • Each element in Y will be between -1,000 and 1,000 inclusive.
  • (X[i], Y[i]) will be pairwise distinct.
