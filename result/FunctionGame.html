Problem Statement for "FunctionGame"

Problem Statement

You are given the ints m and n. Consider the following expression:

Above, F is Euler's totient function. That is, F(n) is the number of positive integers that are &lt;= n and relatively prime to n.

Let X be the value of the above expression. Since the value can be very large, compute and return the value (X modulo 1,000,000,007).

Definition

Class:
FunctionGame
Method:
findValue
Parameters:
int, int
Returns:
int
Method signature:
int findValue(int m, int n)
(be sure your method is public)

Notes

  • You can read more about Euler's totient function here: http://mathworld.wolfram.com/TotientFunction.html
  • Note that F(1) = 1.

Constraints

  • m will be between 1 and 10^6, inclusive.
  • n will be between 1 and 10^9, inclusive.
