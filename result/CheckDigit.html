Problem Statement for "CheckDigit"

Problem Statement

When transmitting data over a network, particularly a wireless network, it is important to verify the integrity of the received data in some way. One common way to do that is to use a check bit or check digit of some sort. In this problem we will use the following algorithm to evaluate a check digit for a String which is known to contain only uppercase letters: 
set runningSum = 0
for each character ch in the string{
    convert the character into a number, where A = 1, B = 2, C = 3, etc.;
    add that number times the one based index of the character to runningSum;
}
return the rightmost digit of runningSum;
Your task is to write a method that implements the above algorithm and returns the check digit for a given String, s.

Definition

Class:
CheckDigit
Method:
getCheckDigit
Parameters:
String
Returns:
int
Method signature:
int getCheckDigit(String s)
(be sure your method is public)

Constraints

  • s will contain between 1 and 50 characters, inclusive.
  • Each character of s will be an uppercase letter ('A'-'Z').
