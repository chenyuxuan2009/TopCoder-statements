Problem Statement for "PowerfulNumbers"

Problem Statement

In this problem you are to find the smallest number, N, such that for each element p of powers, there is some distinct integer k > 1 such that kp is a factor of N. That is, N%(kp) == 0 using a different value of k for each element of powers.

Definition

Class:
PowerfulNumbers
Method:
smallest
Parameters:
int[]
Returns:
long
Method signature:
long smallest(int[] powers)
(be sure your method is public)

Constraints

  • power will contain between 1 and 10 elements greater than 1, inclusive.
  • The return value will fit in a long.
