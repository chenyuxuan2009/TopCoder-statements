Problem Statement for "StringsBetween"

Problem Statement

All strings in this problem are strings of 0 to L lowercase English characters. We will call them valid strings.

All string comparisons are standard comparisons according to their lexicographic order: the string with a smaller character at the leftmost index on which they differ is smaller (e.g., "car" &lt; "cat" because 'r' &lt; 't') and if X is a prefix of Y, X is smaller than Y (e.g., "cat" &lt; "cats").

You are given two valid strings A and B such that A &lt; B.

Return the number of valid strings C such that A &lt; C &lt; B.

Definition

Class:
StringsBetween
Method:
count
Parameters:
int, String, String
Returns:
long
Method signature:
long count(int L, String A, String B)
(be sure your method is public)

Constraints

  • L will be between 0 and 10, inclusive.
  • A and B will be valid strings.
  • A will be less than B.
