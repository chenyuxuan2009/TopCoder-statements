Problem Statement for "MinimumCuts"

Problem Statement

Alice is playing a game with Bob.


The game is played on a binary tree of N vertices, which are numbered from 0 to N-1. (In this problem, a binary tree is any rooted tree in which each vertex has at most two children.) For each integer i such that 0 &lt;= i &lt; N-1, an undirected edge with cost cost[i] connects vertex parent[i] and vertex i+1. If there is no value of i such that parent[i] = X, then vertex X is called a leaf. Bob has a token which starts on vertex 0. At each step of the game:

  • Bob selects an edge which is adjacent to his token.
  • Alice may choose to permanently remove the edge and pay the cost associated with it. If she does so, the move is cancelled.
  • Otherwise, Bob moves the token along the edge.

Bob cannot use any edge more than twice (once in each direction). If BobÃ¢ÂÂs token ever reaches a leaf, Alice loses the game. Alice wins if Bob has no valid moves.


Alice can always win the game. She would like to do so while minimizing the total cost she pays, while Bob wants to maximize it. Assuming both players play optimally, return the cost paid by Alice.
