Problem Statement

Write a program to process the elements of a given string array, get an integer from the process, and return the sum of these integers.

Given class WidgetProcessor:

     * last time you called processName().
     *
     * @return integer between 0 and 16383, inclusive.

DEFINITION

You must define public class WidgetNames with public method int checksum(String[] documents, String delimiterChars, String nameChars).

Each element of documents contains two or more non-overlapping substrings of the form &lt;delimiter&gt;&lt;name&gt;&lt;delimiter&gt;.

Each &lt;delimiter&gt; is any one of the characters found in delimiterChars, while &lt;name&gt; consists of one or more characters found in nameChars. A character cannot appear in both delimiterChars and nameChars.

Instantiate the WidgetProcessor class and obtain a BufferedWriter object by calling getBufferedWriter(). Traverse the elements of documents in order, starting at index 0. From each element, extract the second name. Write the second name to the BufferedWriter, and call processName() to obtain an integer.

If processName() throws a NullPointerException, discard the integer and ignore the remaining elements of documents. If processName() throws any other runtime exception, discard the integer, and move to the next element (if any) of documents.

Otherwise, add the integer to the sum of all previous integers obtained in this manner. Return the final sum.
