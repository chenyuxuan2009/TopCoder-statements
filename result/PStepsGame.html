Problem Statement for "PStepsGame"

Problem Statement

This problem is about a game that is essentially isomorphic to game played in a currently popular TV show. Context has been altered to prevent too large spoilers.


There is a group of N players. The players will play an individual game.

Games are played sequentially: the next player can begin only after the previous player's game has ended.

Each player can observe all previous players. We'll assume that everyone has a perfect memory.


In the game, there are P pairs of bottles. In each pair, one bottle contains water and one contains vinegar.

The player must repeatedly select a bottle and drink a gulp from that bottle. Drinking vinegar immediately eliminates the player from the game.

Once the player has successfully taken P gulps of water from P different bottles (i.e., all available bottles of water), that player wins the game.


Given N and P, calculate and return the expected number of players who will win the game.

PStepsGame

Notes

  • The return value must have an absolute or a relative error at most 1e-9.

Constraints

  • N will be between 1 and 5,000, inclusive.
  • P will be between 1 and 5,000, inclusive.
