Problem Statement for "FindRoot"

Problem Statement

A polynomial is an expression of the form a0x0 + a1x1 + a2x2 + ... + aixi. A root of a polynomial is a value of x such that the the polynomial evaluates to 0. Your task is, given a int[], coeff, where the ith element of coeff represents ai in a polynomial, find the largest root of the polynomial. You are to return the value of this root. If the polynomial has no roots at all, return 0.

Definition

Class:
FindRoot
Method:
largest
Parameters:
int[]
Returns:
double
Method signature:
double largest(int[] coeff)
(be sure your method is public)

Notes

  • Your return must have relative or absolute error of less than 1e-9.

Constraints

  • Each element of coeff will be between -1000 and 1000, inclusive.
  • coeff will contain between 2 and 20 elements, inclusive.
  • At least one element other than element 0 will be non-zero.
  • If the largest root is at x, then the absolute value of the derivative of the polynomial at x will be at least 1e-6.
