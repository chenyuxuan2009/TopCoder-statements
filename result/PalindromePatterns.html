Problem Statement for "PalindromePatterns"

Problem Statement

Using only the letters 'a' and 'b' return how many palindromes of length len do not have forbidden as a substring. A palindrome is a word that is unchanged by reversing it.

Definition

Class:
PalindromePatterns
Method:
howMany
Parameters:
String, int
Returns:
long
Method signature:
long howMany(String forbidden, int len)
(be sure your method is public)

Constraints

  • forbidden will contain between 1 and 5 characters inclusive.
  • len will be between 1 and 100 inclusive.
  • Each character in forbidden must be 'a' or 'b'.
