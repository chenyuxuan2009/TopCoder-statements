Problem Statement for "GoodSequence"

Problem Statement

We say that a sequence of integers is good if, for any non-negative integers i and j such that i &lt; j, the number of occurrences of i in any prefix of the sequence is greater than or equal to the number of occurrences of j. A prefix of a sequence is obtained by removing zero or more elements from the end of the sequence.

You are given a int[] a. Return the length of the longest prefix of a that is good. If there is no good prefix, return 0.

Definition

Class:
GoodSequence
Method:
longestGoodPrefix
Parameters:
int[]
Returns:
int
Method signature:
int longestGoodPrefix(int[] a)
(be sure your method is public)

Constraints

  • a will contain between 1 and 50 elements, inclusive.
  • Each element of a will be between 0 and 100, inclusive.
