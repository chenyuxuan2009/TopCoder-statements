Problem Statement for "WeirdSort"

Problem Statement

You are given a int[] data, and you must order the elements of data such that no two consecutive elements have consecutive values. In other words, result[i] + 1 != result[i + 1] for all i, where 0 &lt;= i &lt; n-1, and n is the number of elements in data. If more than one ordering exists, return the one that comes first lexicographically.

Class:
WeirdSort
Method:
sortArray
Parameters:

Returns:
int[]
Method signature:
int[] sortArray(int[] data)
(be sure your method is public)

Constraints

  • Each element of data will be between 0 and 1000, inclusive.
