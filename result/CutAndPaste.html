Problem Statement for "CutAndPaste"

Problem Statement

You are given the upper left corner and side length of a square selection on a 20 x 20 pixel canvas. You are also given a target location on the same canvas. Execute a cut command on the initial square selection and then execute a paste command at the target location. A cut command puts all the pixels in the selected area into memory and replaces them on the canvas with blank pixels. A paste command places all the pixels in memory onto the canvas, replacing whatever pixels were initially there. The target coordinate is the upper left corner of where the pasted selection will end up. Pixels that lie outside of the 20 x 20 canvas after the paste operation will be cropped. The canvas is given as a String[], canvas, where character j of element i (both 0-indexed) represents the pixel at (i, j). Black pixels are represented as uppercase 'X' characters and blank pixels are represented as '.' characters. The square selection's upper left corner is at (r, c), and its side length is length. The target location's upper left corner is at (targetR, targetC). Return the canvas after the paste command as a String[] in the same format as canvas.

Constraints

  • canvas will contain exactly 20 elements.
  • Each element of canvas will contain exactly 20 characters.
  • Each character in each element of canvas will be either '.' or uppercase 'X'.
  • r, c, targetR, and targetC will each be between 0 and 19, inclusive.
  • length will be between 1 and 20, inclusive.
  • r+length-1 and c+length-1 will each be between 0 and 19, inclusive.
