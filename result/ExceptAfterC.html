Problem Statement for "ExceptAfterC"

Problem Statement

English spelling is full of crazy rules. One rule that children learn is "I before E except after C", meaning that whenever an 'I' and an 'E' are adjacent, the 'I' must precede the 'E' unless the pair is immediately after a 'C', in which case the 'E' must precede the 'I'.

Write a method that checks compliance with this rule. The method takes a String word and returns the number of times the rule is violated in word. Notice that a single letter can be involved in more than one violation of the rule (see Example 4).

Definition

Class:
ExceptAfterC
Method:
violations
Parameters:
String
Returns:
int
Method signature:
int violations(String word)
(be sure your method is public)

Constraints

  • word contains between 1 and 25 characters, inclusive.
  • Every character in word is an uppercase letter ('A'-'Z', inclusive).

Examples

  1. "TOPCODER"

    Returns: 0

    No opportunity to go wrong.

  2. "EI"

    Returns: 1

    The simplest error case.

  3. "CIE"

    Returns: 1

  4. "PIES"

    Returns: 0

  5. "CEI"

    Returns: 0

  6. "ICE"

    Returns: 0

    Be careful at end of string.

  7. "RECIEVE"

    Returns: 1

  8. "IEIEIEEECCCIEIEIIIEEIEIE"

    Returns: 7

  9. "AEIOUIECEIECIEE"

    Returns: 2

  10. "RECEIVE"

    Returns: 0

  11. "NEIGHBOR"

    Returns: 1

    In this simplified version of "I before E except after C", ignore the usual extension "or when pronounced as A". So the "EI" in "NEIGHBOR" is an error for the purposes of this problem, even though it is okay in English.

  12. "E"

    Returns: 0

  13. "ACIEIB"

    Returns: 2

    The 'E' is involved in two separate violations: first in the "IE" after the 'C', and then in the "EI".

  14. "EIEIEICIEIEIE"

    Returns: 6

  15. "C"

    Returns: 0

  16. "I"

    Returns: 0

  17. "X"

    Returns: 0

  18. "IE"

    Returns: 0

  19. "CI"

    Returns: 0

  20. "CE"

    Returns: 0

  22. "EIEUIEWCCEEREZECIEIICITNC"

    Returns: 3

  23. "CCCIIZCUIICCCIHCNNGCICICG"

  25. "XSLCEEJCCECUECIICEICGIUIJ"

    Returns: 0

  26. "IEEERCCEICCITECEDECCZIECC"

  28. "ECIIIIEEOEWCTICCIPIIFEECC"

    Returns: 0

  29. "EICCCRCIIRIICIECCRECCIECC"

    Returns: 3

  30. "ACAICEBEILCIDICFCCLRTDIYE"

    Returns: 1
