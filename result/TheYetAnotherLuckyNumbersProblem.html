Problem Statement for "TheYetAnotherLuckyNumbersProblem"

Problem Statement

John and Brus believe that the digits 4 and 7 are lucky and all others are not. According to them, a lucky number is a number that contains only lucky digits in its decimal representation.

You are given longs A, B and C. Return the number of integers between A and B, inclusive, that can be represented as a sum of up to C lucky numbers.

Definition

Class:
TheYetAnotherLuckyNumbersProblem
Method:
find
Parameters:
long, long, long
Returns:
long
Method signature:
long find(long A, long B, long C)
(be sure your method is public)

Constraints

  • A will be between 1 and 474747474747474747, inclusive.
  • B will be between A and 474747474747474747, inclusive.
  • C will be between 1 and 7, inclusive.
