Problem Statement for "VertHorizInter"

Problem Statement

You will be given a list of line segments in the plane. A line from point (x1,y1) to (x2,y2) will be denoted by the String (quotes for clarity): "x1 y1 x2 y2". Each line given will be vertical or horizontal. No two lines will intersect at more than a single point. You will return a String[] containing the intersection points. Each element of the returned list will represent a single point. Each point returned should be formatted as (quotes for clarity) "x y" where x and y are the x and y coordinates of the point, respectively. x and y must be separated by a single space. All integers in this problem (input or returned) should have no extra leading zeros. The returned points should be sorted in increasing order by x-coordinate. If two points have the same x-coordinate, sort by increasing y-coordinate.

Class:
VertHorizInter
Method:

Parameters:

Returns:

String[] howMany(String[] lines)
(be sure your method is public)

Constraints

  • lines will contain between 2 and 50 elements inclusive.
  • Each element of lines will contain between 7 and 19 characters inclusive.
  • Each element of lines will be formatted as "x1_y1_x2_y2" where'_' denotes a single space.x1,y1,x2, and y2 denote integers with no extra leading zeros between 0 and 1000 inclusive.
  • Each element of lines will contain no leading or trailing whitespace.
  • No element of lines will be a single point (the endpoints for a given line will be distinct).
  • Each element of lines will be either a horizontal or vertical line.
  • The intersection of any two elements of lines will be at most a single point.
