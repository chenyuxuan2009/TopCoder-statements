Problem Statement for "ShippingCost"

Problem Statement

Calculate how much it would cost to ship a box given its weight and dimensions. The shipping service charges rate dollars for each pound that a package weighs. However, if a package weighs less than thresholdWeight pounds and its (length + girth) is greater than thresholdSize inches, it is billed as a package that weighs thresholdWeight pounds. The length of a box is its longest side. The girth is the distance around the box perpendicular to the length. In other words, if the two non-length sides are called width and height, girth is defined as width + height + width + height. If a box has two equal sides that are greater than the third side, or if a box has three equal sides, any of the multiple longest sides can be used as the length.

The dimensions of the box are given as a String formatted as "<side>x<side>x<side>" (quotes for clarity only), where each <side> is a positive integer with no leading zeros. All lengths are given in inches, and all weights are given in pounds.

Constraints

  • dimensions will be formatted as "xx" (quotes for clarity only), where each is an integer between 1 and 100, inclusive, with no leading zeros. Each 'x' is lowercase.
  • weight will be between 1 and 100, inclusive.
  • rate will be between 1 and 10, inclusive.
  • thresholdSize will be between 20 and 300, inclusive.
  • thresholdWeight will be between 30 and 100, inclusive.
