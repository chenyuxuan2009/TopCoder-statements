Problem Statement for "BeautifulBanner"

Problem Statement

You are so excited about coming to TechOlympics that you've decided to prepare a huge and beautiful banner with "TechOlympics" written on it. However, there is a little problem: you're not really good at drawing. Fortunately, there is a little shop nearby that sells nice individual paper letters. So your plan is to go to the shop, buy proper letters, and glue them onto a sheet of paper to form the word "TechOlympics".

There are N letters currently available in the shop. They are described by the String letters and the int[] beauty. For each i, 0 &lt= i &lt; N-1, there is one letter given by the i-th character of letters and its beauty value is beauty[i].

The beauty value of the banner is the sum of beauty values of individual letters used for the banner. Return the maximum beauty value of a "TechOlympics" banner that you can create from the given letters. If no banner can be created, return -1 instead. Note that lowercase and uppercase letters are considered different in this problem.

Definition

Class:
BeautifulBanner
Method:
maximumBeauty
Parameters:
String, int[]
Returns:
int
Method signature:
int maximumBeauty(String letters, int[] beauty)
(be sure your method is public)

Constraints

  • letters will contain between 10 and 50 characters, inclusive.
  • Each character in letters will be a lowercase letter ('a'-'z') or an uppercase letter ('A'-'Z').
  • beauty will contain the same number of elements as the number of characters in letters.
  • Each element of beauty will be between 0 and 100, inclusive.
