Problem Statement for "BitOrder"

Problem Statement

Sort all the integers between first and last, inclusive, using the number of 1's in their binary representations. Integers with fewer 1's must come before integers with more 1's. If two integers have the same number of 1's, the smaller integer must come first. Return the k-th (0-based) integer in the sorted list.

Definition

Class:
BitOrder
Method:
kthElement
Parameters:
int, int, int
Returns:
int
Method signature:
int kthElement(int first, int last, int k)
(be sure your method is public)

Constraints

  • first will be between 1 and 10^9, inclusive.
  • last will be between first and 10^9, inclusive.
  • k will be between 0 and (last - first), inclusive.

    Returns: 8
