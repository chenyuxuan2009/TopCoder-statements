Problem Statement for "TrainTickets"

Problem Statement

John is buying train tickets for a group of people. He gave several banknotes to the cashier, but forgot to mention the exact number of tickets he needs. The cashier knows that John may want to buy K tickets only if K satisfies the following conditions:

  1. The amount given to the cashier is enough to pay for K tickets.
  2. John did not give any extraneous banknotes. In other words, if he were to take back any single banknote, he would no longer be able to pay for K tickets.

A int[] banknotes contains information about the banknotes John gave to the cashier, where banknotes[i] represents the value of the i-th banknote. Also given is the price of a single ticket. Return the number of possible values for K.

Definition

Class:
TrainTickets
Method:
howManyWays
Parameters:
int, int[]
Returns:
int
Method signature:
int howManyWays(int price, int[] banknotes)
(be sure your method is public)

Constraints

  • price will be between 1 and 100, inclusive.
  • banknotes will contain between 1 and 50 elements, inclusive.
  • Each element of banknotes will be between 1 and 100, inclusive.
