Problem Statement for "SumOfTerms"

Problem Statement

There is a recurrent sequence F. Its first T terms are F[i]=firstTerms[i], for i from 0 through T-1. For i &gt;= T, F[i] is given by the following recurrence relation:

F[i] = sum (for j from 0 through T-1) coefficients[j]*F[i+j-T]

You're given two int[]s firstTerms and coefficients, containing T elements each, and one long n. Compute and return the sum of the first n terms of sequence F, modulo 1,000,000,007.

Definition

Class:
SumOfTerms
Method:
find
Parameters:
int[], int[], long
Returns:
int
Method signature:
int find(int[] firstTerms, int[] coefficients, long n)
(be sure your method is public)

Constraints

  • coefficients and firstTerms will each contain T elements.
  • T will be between 1 and 50, inclusive.
  • Each element of firstTerms will be between 1 and 1,000,000, inclusive.
  • Each element of coefficients will be between 1 and 1,000,000, inclusive.
  • n will be between T and 1,000,000,000,000,000,000 (10^18), inclusive.
