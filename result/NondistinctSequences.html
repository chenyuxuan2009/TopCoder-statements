Problem Statement for "NondistinctSequences"

Problem Statement

A sequence of integers is called k-nondistinct if, for every k consecutive elements, there are at least two equal numbers. For example, the sequence {1, 2, 1, 2} is 3-nondistinct, but the sequence {1, 2, 3, 2} is not.

You will be given ints n, k and m. Return the number of sequences containing exactly n elements, where each element is between 1 and m, inclusive, that are k-nondistinct. Since this number can be quite large, return it modulo 1,000,000,007.

Definition

Class:
NondistinctSequences
Method:
count
Parameters:
int, int, int
Returns:
int
Method signature:
int count(int n, int k, int m)
(be sure your method is public)

Constraints

  • n will be between 1 and 50, inclusive.
  • k will be between 2 and 7, inclusive.
  • m will be between 1 and 50, inclusive.
