Problem Statement

You've collected some data, and now you want to generate a histogram. Each data point is a floating point number between 0 and 1, exclusive. Your histogram should have N bars, each representing an interval of the same size. For example, if N=4, the bars of the histogram would represent the ranges 0.0-0.25, 0.25-0.5, 0.5-0.75 and 0.75-1.0. Given a String[], data, where each element represents a decimal value, return a int[], with N elements, representing the N intervals, in order. Each element of the return should be equal to the number of elements of data that fall into the corresponding interval. You need not worry about data points that fall exactly on the boundary of an interval, as that will not occur.

Constraints

  • data will contain between 0 and 50 elements, inclusive.
  • Each element of data represents a decimal value between 0 and 1, exclusive.
  • Each element of data will be formatted as a "0." followed by a sequence of between 1 and 20 integers.
  • No element of data will fall within 1e-9 of the boundary of two intervals.
  • N will be between 1 and 50, inclusive.
