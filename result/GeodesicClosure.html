Problem Statement for "GeodesicClosure"

Problem Statement

Given a directed graph g, and a set s of nodes in that graph, determine the minimum sized subgraph such that for all pairs of nodes (a,b) in s, if there was a shortest path of length k from a to b in the original graph, there is still a shortest path of length k from a to b in the subgraph.

g will be a square String[] (same number of characters per element as elements), where character i of element j is '1' if there is a directed edge from j to i, and '0' otherwise.

Definition

Class:
GeodesicClosure
Method:
size
Parameters:
String[], int[]
Returns:
int
Method signature:
int size(String[] g, int[] s)
(be sure your method is public)

Notes

  • There is always a path of length 0 from a node to itself.
  • G' is a subgraph of G if the nodes in G' are a subset of the nodes in G, and the edges in G' are a subset of the edges in G. The size of any graph is simply the number of nodes.

Constraints

  • g will have between 1 and 20 elements, inclusive.
  • g will be square, and each character will be '0' or '1'.
  • s will have between 1 and the number of elements in g, inclusive.
  • Each element of s will be between 0 and the number of elements in g - 1, and s will contain no duplicates.
  • There are no edges of the form (i,i).
