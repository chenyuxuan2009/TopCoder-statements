Problem Statement for "ManyHashes"

Problem Statement

Suppose you have a string S of digits, with each digit between 0 and 2 inclusive. We can treat S as a number in ternary (base 3). Ternary is similar to binary, except in ternary the digits represent powers of 3 instead of powers of 2. We compute the hash of S as follows: 
	integer hash = 0
	for (integer pos = 0 to LengthOfS-1 inclusive) {
		hash = hash + pos*pos*(value of digit in position pos of S)
        }
	hash = hash + (value of S interpreted as a number in ternary)
The value of the digit 0 is 0, 1 is 1, and 2 is 2. Given an int len, return the sum of the hashes of all strings of length len composed of 0's, 1's, and 2's. Note that these strings may have leading zeros.
