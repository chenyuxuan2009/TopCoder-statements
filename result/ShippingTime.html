Problem Statement for "ShippingTime"

Problem Statement

You are shipping a box from sourceZipCode to destZipCode. Determine how long it will take for the package to arrive at its destination. You are given a String[], zones, which defines the locations of various zip codes. Each element of zones is a single space delimited list of five-digit integers. Element i (zero-indexed) of zones contains all the zip codes located in zone i. You are also given a int[], times, where the ith element (zero-indexed) is the number of days it takes for a package to travel between two zones whose difference is i. The difference between zone z1 and zone z2 is defined as the absolute value of z1 minus z2.

Definition

Class:
ShippingTime
Method:
getTime
Parameters:
int, int, String[], int[]
Returns:
int
Method signature:
int getTime(int sourceZipCode, int destZipCode, String[] zones, int[] times)
(be sure your method is public)

Constraints

  • sourceZipCode and destZipCode will each be between 10000 and 99999, inclusive.
  • zones will contain between 1 and 50 elements, inclusive.
  • Each element of zones will contain between 5 and 47 characters, inclusive.
  • Each element of zones will be a single space delimited list of one or more five-digit integers, with no leading or trailing spaces.
  • zones will contain no duplicate integers.
  • Both sourceZipCode and destZipCode will exist in zones.
  • times will contain the same number of elements as zones.
  • Each element of times will be between 1 and 7, inclusive.
