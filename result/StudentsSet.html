Problem Statement for "StudentsSet"

Problem Statement

Write a utility to sort students based on their characteristics.

Given class StudentsUtils:

Contains various methods to solve this problem.

class StudentsUtils {
  /**
   *  Creates a StudentsUtils object corresponding to the given data seed.
   * 
   *  @param dataSeed is an integer between 0 and 10^9, inclusive.
   */
  public StudentsUtils(int dataSeed);

  /**
   *  Returns the set of students to sort. All students in the set will have distinct names and IDs.
   * 
   *  @return Set containing between 1 and 10,000 students, inclusive.
   */
  public Set&lt;Student&gt; getStudentsSet();

  /**
   *  Gets the array containing IDs of various students. This array is sorted in
   *  ascending order and must have IDs of all students from getStudentsSet().
   *  All IDs in the array are distinct.
   * 
   *  @return Array containing between 1 and 1,000,000 integers, inclusive, where each
   *          integer is between 0 and 10^9, inclusive.
   */
  public int[] getStudentsIds();

  /**
   *  Gets the array containing the same number of elements as getStudentsIds(). The i-th
   *  element of this array denotes marks corresponding to subject 1 of the student with the ID given
   *  in the i-th element of array getStudentsIds().
   *
   *  @return Array containing between 1 and 1,000,000 integers, inclusive, where each
   *          integer is between 0 and 10, inclusive.
   */
  public int[] getStudentsMarks1();

  /**
   *  Gets the mapping of students to their marks for subject 2. The set of keys
   *  in this map contains exactly the same students as getStudentsSet().
   * 
   *  @return A Map&lt;Student, Integer&gt; where the value for each key is an integer
   *          between 0 and 10, inclusive.
   */
  public Map&lt;Student, Integer&gt; getStudentsMarks2();

  /**
   *  Gets the check sum for the given SortedSet&lt;Student&gt;. This checksum depends
   *  on the contents of the set and the order of students.
   *
   *  @param students is a sorted set containing students.
   *  @return the checksum for the given sorted set of students.
   */
  int checkStudentsOrder(SortedSet&lt;Student&gt; students);
}

Given class Student:

Describes a single student. Each student has a unique identifier, grade and name. For some students, you can also get their marks for two fixed subjects by using the methods of class StudentsUtils.

class Student {
  /**
   *  Get the unique identifier for this student.
   * 
   *  @return Identifier between 0 and 10^9 inclusive.
   */
  int getId();

  /**
   *  Gets the grade for this student.
   * 
   *  @return Grade between 0 and 10, inclusive.
   */
  int getGrade();

  /**
   *  Gets the name of this student.
   * 
   *  @return String containing between 1 and 20 letters ('a'-'z', 'A'-'Z'), inclusive.
   */
  String getName();
}

DEFINITION

You must define a public class StudentsSet with public method int orderStudents(int dataSeed).

Implement orderStudents as follows:

Create StudentsUtils stUtils by passing dataSeed to StudentsUtils(int). Obtain the set of students to sort using stUtils.getStudentsSet(). Get the marks for subjects 1 and 2 for all students in this set. Then, sort all the students according to the following rules:

  1. If two students A and B have distinct grades, the student with the smaller grade is lower than the other student. Otherwise, go to step 2.
  2. If two students A and B have distinct marks for the subject 1, the student with the lesser marks is lower than the other student. Otherwise, go to step 3.
  3. If two students A and B have distinct marks for the subject 2, the student with the lesser marks is lower than the other student. Otherwise, go to step 4.
  4. The student with alphabetically smaller name is lower than the other student.

Create a SortedSet&lt;Student&gt; containing all the students from getStudentsSet() sorted in the described order. Pass this sorted set to checkStudentsOrder() and return the resulting integer.
