Problem Statement for "InternetProvider"

Problem Statement

You are choosing a single internet provider from a list. If you choose the ith provider, you must pay prepaid[i] cents every month, which allows you to use prepaidLimit[i] MB each month. You must pay an additional costPerMB[i] cents for every MB which exceeds the prepaid monthly limit.

Supposing your monthly traffic is equal to monthlyTraffic MB, calculate the minimal monthly cost in cents for this traffic.

Definition

Class:
InternetProvider
Method:
minimumPayment
Parameters:
int[], int[], int[], int
Returns:
int
Method signature:
int minimumPayment(int[] prepaid, int[] prepaidLimit, int[] costPerMB, int monthlyTraffic)
(be sure your method is public)

Constraints

  • prepaid, prepaidLimit and costPerMB will have the same number of elements.
  • prepaid will contain between 1 and 50 elements, inclusive.
  • Each element of prepaid will be between 0 and 10000, inclusive.
  • Each element of prepaidLimit will be between 0 and 10000000, inclusive.
  • Each element of costPerMB will be between 0 and 20, inclusive.
  • monthlyTraffic will be between 0 and 1000000, inclusive.
