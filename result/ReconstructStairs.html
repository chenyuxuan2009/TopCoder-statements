Problem Statement for "ReconstructStairs"

Problem Statement

There once was a perfect staircase made of unit cubes. The staircase went from the left to the right. The first step was just a single cube, and each following step was built using one more cube than the previous one.

An example of such a staircase is shown below: 


      #
     ##
    ###
   ####
  #####
 ######

(This is a 6-step staircase. It is built using 1+2+3+4+5+6 unit cubes.)

The perfect staircase that once existed has been damaged over time: from some columns some cubes disappeared. Of course, gravity still works, so in each column the remaining cubes are still on the bottom.

You are given the heights of all the remaining columns, from the left to the right, in the int[] columns. For example, columns = { 3, 1, 0, 4, 0, 0, 2 } represents the following configuration of cubes: 


     #
  #  #
  #  #  #
  ## #  #
  3104002

You were given the task to repair the staircase: add some (possibly zero) new cubes so that the result will be a perfect staircase that goes to the right.

The staircase must contain all the existing cubes. The existing cubes cannot be moved from their current locations.

You can choose the size and the exact position of your staircase. While building it, you are allowed to add completely new columns of cubes (see Example #3).

Return the minimum number of additional cubes needed to repair the given staircase.

Definition

Class:
ReconstructStairs
Method:
reconstruct
Parameters:
int[]
Returns:
int
Method signature:
int reconstruct(int[] columns)
(be sure your method is public)

Constraints

  • columns will contain between 1 and 50 elements, inclusive.
  • Each element of columns will be between 0 and 50, inclusive.
  • Sum of columns will not be zero.
