Problem Statement for "TripleString"

Problem Statement

A string is called a triple if it is a concatenation of three copies of another string. For example, "abbabbabb", "www", "testtesttest" are triple strings, while "aaaa" and "ababac" are not.

You are given a String s. The length of s is divisible by 3. You are allowed to choose any number of letters in s and replace each of them with a different letter. Return the minimal number of letters you must change to transform s into a triple string.

Definition

Class:
TripleString
Method:
minChanges
Parameters:
String
Returns:
int
Method signature:
int minChanges(String s)
(be sure your method is public)

Constraints

  • s will contain between 3 and 48 characters, inclusive.
  • The number of characters in s will be divisible by 3.
  • s will contain only lowercase letters ('a'-'z').

    Returns: 3
