Problem Statement for "AcyclicBuilder"

Problem Statement

You are given a directed graph as a String[].The j-th character of the i-th element of graph is '1' (one) if there is a directed edge from the i-th vertex to the j-th vertex, and '0' (zero) if there is not. Return the minimal number of edges you need to delete to make it acyclic.

Definition

Class:
AcyclicBuilder
Method:
minEdgeCount
Parameters:
String[]
Returns:
int
Method signature:
int minEdgeCount(String[] graph)
(be sure your method is public)

Notes

  • The graph may contain loops, i.e. edges from a vertex to itself. Each such edge forms a cycle. See example 1 for clarity.

Constraints

  • graph will contain between 1 and 21 elements, inclusive.
  • Each element of graph will contain exactly n characters, where n is the number of elements in graph.
  • Each element of graph will contain only the digits '0', '1'.
