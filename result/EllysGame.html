Problem Statement for "EllysGame"

Problem Statement

Elly is playing a new game. On a rectangular board with N rows and M columns are located several spaceships. Each spaceship can move in a single (pre-defined) direction with a speed of one cell per second. It moves until either going beyond the limits of the board (in which case the game ends), or reaches another ship, in which case it crashes into it and disappears. After the crash the other ship starts moving (in its own direction).

The initial board is given to you in the String[] board. Each symbol in board is one of {'.', '&lt;', '&gt;', '^', 'v'}. A '.' denotes an empty cell, and each of the others - a spaceship. The spaceships move in the direction their symbol points to.

Initially none of the spaceships are moving. All the player does is make a single selection - for the first ship to start moving. Depending on which ship is selected, the game can continue various time. Now Elly wonders what is the maximal number of seconds the game can last, if choosing the optimal ship. Please help the girl by finding this amount of time.

Notes

  • Due to the way the game is played it is guaranteed it will finish in a finite numeber of seconds.

Constraints

  • board will contain between 1 and 50 elements, inclusive.
  • Each element of board will contain between 1 and 50 characters, inclusive.
  • All elements of board will contain the same number of elements.
  • Each character in board will be from the alphabet {'.', '<', '^', '>', 'v'}.
  • There will be at least one character, different than '.' in board.
