Problem Statement for "KnightAttack"

Problem Statement

You are given a 8x8 chessboard containing a knight. A knight moves two cells in one of four directions (up, down, left or right), and then one more cell in a perpendicular direction. It must always remain on the chessboard. Some of the cells on the chessboard are occupied, and the knight is not allowed to land in any of those cells. The knight is initially standing on a free cell.

8 .N......
7 ........
6 ........
5 ........
4 ..xxx...
3 ..x.x...
2 ..xxx...
1 ........
  abcdefgh

The 'N' character represents the knight, 'x' characters represent occupied cells, and '.' characters represent free cells. In the example above, cell d3 can be visited in 3 moves.

You are given a String[] chessboard containing the initial layout of the board. Element 0 of chessboard contains cells a8, b8, c8, ..., h8; element 1 contains cells a7, b7, c7, ..., h7; and so on. You are also given a String[] queries. Each element of queries is formatted as "cr K" (quotes for clarity), where c and r are the column and row, respectively, of a cell, and K is an integer. Columns are labelled 'a' to 'h' (left to right) and rows are numbered '1' to '8' (bottom to top). Each element of queries represents a single question: Can the knight reach the cell at column c, row r in exactly K moves from its initial starting position? Note that the knight is allowed to revisit free cells (including its starting and destination cells) multiple times. Return a String that contains exactly n characters, where n is the number of elements in queries. The i-th character of the return value should be 'Y' if the answer to the i-th element of queries is yes, and 'N' otherwise.

Constraints

  • chessboard will contain exactly 8 elements and each element will contain exactly 8 characters.
  • Each element of chessboard will contain only 'N', '.' or 'x' characters.
  • There is exactly one 'N' character in chessboard.
  • queries will have contain between 1 and 50 elements, inclusive.
  • Each element of queries will be formatted as "cr K" (quotes for clarity), where c is a lowercase letter between 'a' and 'h', inclusive, r is a digit between '1' and '8', inclusive, and K is an integer between 1 and 1000000000, inclusive, with no leading zeroes.
  • Each queried cell will be free.
