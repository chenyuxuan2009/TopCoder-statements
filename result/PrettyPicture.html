Problem Statement for "PrettyPicture"

Problem Statement

We want to take a picture of the night sky.

For the purpose of this problem, the night sky is a rectangular array of cells with R rows and C columns. There is at most one star in each cell.

Our camera can take a shot of any rectangular area of the sky. The area in the picture will cover exactly PR consecutive rows and PC consecutive columns of the sky. Calculate and return the maximum number of stars in the picture.

Use the following pseudocode to generate the night sky: 


state = seed
for cellr = 0 to R-1:
    for cellc = 0 to C-1:
        state = (state * 1103515245 + 12345) modulo 2^31
        if state &lt; threshold:
            cell with coordinates (cellr, cellc) contains a star
        else:
            cell with coordinates (cellr, cellc) is empty

Definition

Class:
PrettyPicture
Method:
takePicture
Parameters:
int, int, int, int, int, int
Returns:
int
Method signature:
int takePicture(int R, int C, int PR, int PC, int seed, int threshold)
(be sure your method is public)

Notes

  • The reference solution does not depend on the sky being pseudorandom, it would correctly solve any input of the size allowed by the Constraints.

Constraints

  • 1 <= PR <= R <= 4000
  • 1 <= PC <= C <= 4000
  • seed will be between 0 and 2^31 - 1, inclusive.
  • threshold will be between 0 and 2^31 - 1, inclusive.
