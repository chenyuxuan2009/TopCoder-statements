Problem Statement for "TennisTournament"

Problem Statement

You are organizing a tennis tournament. There are several players and they need to be paired up to participate. Each player has a level indicated by an integer value from 1 to 100. The level of a pair is the sum of the levels of both players. Of course, each player cannot be assigned to more than one pair. Also, for the tournament to be fun, you decided that all pairs needed to be of the same level. If necessary, some players may be rejected from the tournament. Of course, you want to minimize the number of rejected players.

You will be given the levels of all players as a int[]. Return the minimum number of players that need to be rejected.

Definition

Class:
TennisTournament
Method:
maxPairs
Parameters:
int[]
Returns:
int
Method signature:
int maxPairs(int[] level)
(be sure your method is public)

Constraints

  • level will contain between 1 and 50 elements, inclusive.
  • Each element of level will be between 1 and 100, inclusive.
