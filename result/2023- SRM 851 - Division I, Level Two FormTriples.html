Problem Statement for "FormTriples"

Problem Statement

There are three character types: digits, lowercase letters and uppercase letters. Digits are D = "0123456789". Lowercase letters are L = "abcdefghij". Uppercase letters are U = "ABCDEFGHIJ".

(We will use just the first 10 letters of the English alphabet.)

Within each type, the individual characters have indices corresponding to their 0-based index in the corresponding string given above. For example, the index 2 corresponds to the digit '2', the lowercase letter 'c', and the uppercase letter 'C'.


You are given the int N, meaning that we will only be using the first N characters of each type.

You are also given the int C containing N positive integers. For each i and each character type X, take exactly C[i] copies of the character X[i].

Divide all characters you took in the previous step into triples (digit, lowercase letter, uppercase letter) in such a way that within each triple the three characters have three mutually distinct indices.

Concatenate each triple (in the order given above) and then concatenate all triples to form one long string. Return that string. If there are multiple solutions, you may return any one of them. If there are no solutions, return an empty string instead.

Constraints

  • N will be between 1 and 10, inclusive.
  • C will have exactly N elements.
  • All elements of C will be positive.
  • sum(C) will not exceed 1000.
