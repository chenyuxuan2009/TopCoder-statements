Problem Statement for "NewTimeSystem"

Problem Statement

It was recently decided that time would be measured using a 10-based system. The length of a day remains the same, but it now contains 100 "new" hours (instead of 24 "old" hours). Each "new" hour contains 100 "new" minutes, and each "new" minute contains 100 "new" seconds. You must now write a program that converts "old" times into "new" times.

You will be given a String time containing a time written in the form "HH:MM:SS" (quotes for clarity), where HH, MM and SS are hours, minutes and seconds in the "old" time system. Return the same time in the same format, but in the "new" 10-based time system. Any fractional seconds should be omitted.
