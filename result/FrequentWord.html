Problem Statement

Given a list of strings, your method should find the word that occurs most frequently in the list. The given strings will be single space delimited lists of words. This means (quotes for clarity) "abba" is equivalent to "AbbA". Your method will return the number of times the most frequently used word occurs.

FrequentWord
Method:

Parameters:
String[]
Returns:
int
Method signature:
int howMany(String[] wordlist)
(be sure your method is public)

Constraints

  • Each element of wordlist will contain between 1 and 50 characters inclusive.
  • Each element of wordlist will contain no leading or trailing whitespace.
  • Each element of wordlist will contain only lowercase letters ('a'-'z'), uppercase letter ('A'-'Z'), or spaces (' ').
  • The elements of wordlist will be single space delimited lists of words. Each word will contain only letters.
