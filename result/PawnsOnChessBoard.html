Problem Statement for "PawnsOnChessBoard"

Problem Statement

You are given a String[] chessBoard representing a chess board, where the c-th character of the r-th element represents the square at row r, column c. An empty square is represented by '.' and an occupied square is represented by '#'. All coordinates in this problem are given as (row, column). Your task is to place pawns on the board using the following algorithm: 

for each row r from first to the last
  for each column c from first to the last
    if square at position (r, c) is empty and 
    not attacked by pawn
      place pawn on square at position (r, c)

A pawn at square (r, c), where r is not the last row, attacks square (r + 1, c + 1) (if c is not the last column) and also square (r + 1, c - 1) (if c is not the first column).

Return the number of pawns that are placed on the board using this algorithm.

Constraints

  • chessBoard will contain between 1 and 50 elements, inclusive.
  • Each element of chessBoard will contain between 1 and 50 characters, inclusive.
  • All elements of chessBoard will be of the same length.
  • Each character of each element of chessBoard will be either '.' or '#'.
