Problem Statement for "DismantleTheTree"

Problem Statement

We have a weighted tree with N nodes, numbered 1 through N. You are given the description of the tree: the int[]s X, Y and edgeWeight, each with N-1 elements. For each i between 0 and N-2, inclusive, there is an edge that connects the nodes X[i] and Y[i], and the weight of this edge is edgeWeight[i].

You want to erase all edges of the tree as cheaply as possible. In each step, you can select any simple path that consists of edges of the same weight W, pay W, and erase all the edges that form the path.

(Note that you only pay W once, regardless of the length of the path. Also note that the entire path must still exist in the tree.)

Compute and return the minimum total cost you need to pay to erase all edges of the given tree.

Definition

Class:
DismantleTheTree
Method:
dismantle
Parameters:
int, int[], int[], int[]
Returns:
int
Method signature:
int dismantle(int N, int[] X, int[] Y, int[] weight)
(be sure your method is public)

Constraints

  • N will be between 2 and 1000, inclusive.
  • X, Y, and edgeWeight will each contain exactly N-1 elements.
  • All elements of X and Y will be between 1 and N, inclusive.
  • All elements of edgeWeight will be between 1 and 100, inclusive.
  • It is guaranteed that the input will be a tree.
