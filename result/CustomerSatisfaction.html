Problem Statement

In an attempt to improve the quality of its inventory, a store would like to remove the products that have the lowest customer satisfaction rates. It will do this by checking its sales logs and determining which products have the highest return rates. Product information will be given as a String[], products, where each element represents a single product and is formatted as (quotes and brackets for clarity only): "<PRODUCT_ID> <PRODUCT_NAME>", where <PRODUCT_ID> is an integer which uniquely identifies a product, and <PRODUCT_NAME> is the name of the product. Sales records will be given as a String[], sales, where each element represents the sales summary of a single product and is formatted as (quotes and brackets for clarity only): "<PRODUCT_ID> <QTY_SOLD> <QTY_RETURNED>", where <PRODUCT_ID> is an integer which corresponds directly to the <PRODUCT_ID>s in products, <QTY_SOLD> is the total number of units sold, and <QTY_RETURNED> is the total number of units returned. Given an int, threshold, return the names of all the products that have a return rate of at least threshold percent. The product names should be returned as a String[] sorted in alphabetical order. If no products meet the criteria, return an empty String[].
