Problem Statement

You have a number of tasty ingredients and you want to know how many different banana split desserts you could make using those ingredients. In order for something to count as a banana split, it must have at least one &quot;banana&quot;, at least one &quot;ice cream&quot;, and at least one other ingredient (i.e., at least one ingredient that is not a "banana" or an "ice cream"). Ingredients with the same name are not distinguishable, and it doesn't matter what order the items are added to the dessert. For example, suppose you had the following 5 ingredients:
  • &quot;ice cream&quot;,&quot;banana&quot;,&quot;chocolate&quot;,&quot;chocolate&quot;,&quot;peanuts&quot;
You could make a total of 5 different banana splits, as shown below:
  • &quot;ice cream&quot;,&quot;banana&quot;,&quot;chocolate&quot;,&quot;chocolate&quot;,&quot;peanuts&quot;
  • &quot;ice cream&quot;,&quot;banana&quot;,&quot;chocolate&quot;,&quot;peanuts&quot;
  • &quot;ice cream&quot;,&quot;banana&quot;,&quot;chocolate&quot;,&quot;chocolate&quot;
  • &quot;ice cream&quot;,&quot;banana&quot;,&quot;chocolate&quot;
  • &quot;ice cream&quot;,&quot;banana&quot;,&quot;peanuts&quot;
Note that if you would have had 2 bananas, you would have had 10 possibilities - as adding a banana to each combination above makes a new dessert. Return the number of distinct banana splits you could create using the items given in ingredients.

  • Each element of ingredients will contain only lowercase letters and spaces ('a'-'z',' ').
