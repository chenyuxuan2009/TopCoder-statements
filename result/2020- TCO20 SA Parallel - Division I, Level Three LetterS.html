Problem Statement

A letter S shape is a set of cells in a square grid that can be painted as follows.

  1. Paint a horizontal contiguous segment that consists of at least two cells. Let L and R be the leftmost and the rightmost of these cells.
  2. Starting at L, paint a vertical contiguous segment going upwards. In addition to L this segment must contain at least two more cells. Let the topmost of those cells be T.
  3. Starting at T, paint a horizontal contiguous segment going to the right. In addition to T this segment must contain at least one more cell.
  4. Similarly to the previous two steps, paint a vertical segment going down from R to B (at least two new cells) and then from B going left (at least one new cell).

You are given a grid with R rows and C columns. Both rows and columns are numbered starting from zero. This grid contains some obstacles: for each valid i, there is an obstacle blocking the cell in row obr[i], column obc[i]. Cells with obstacles cannot be painted.

Count all ways in which a letter S shape can be painted into the given grid. Return that count modulo 10^9 + 7.

  • The cells described by obr and obc will be distinct.
