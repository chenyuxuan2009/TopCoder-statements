Problem Statement

NOTE: You may only submit each problem one time. If you submit more than once, your submissions may not count.

It's time to trade in that old 486 and get a faster computer. But, there are so many different choices on the market today, it is difficult to decide which machine to get. You have decided that there are 4 factors that really matter to you: CPU speed, memory size, hard disk size, and price. You would like a computer with a higher speed, a larger amount of memory, a larger hard disk, and a lower price.

You are not sure exactly which of those four factors is most important, but you know one thing for sure: if computer A has a higher speed, more memory and a larger hard disk than computer B, and is also cheaper than computer B, then computer A is definitely better than computer B. Also, if computer A is better than computer B with regard to some of those characteristics, and equal to computer B in the rest of them, then computer A is also definitely better than computer B.

For example, if computer A has a speed of 1000, a memory of size 128, a hard disk of size 80, and costs 500, it is definitely better than computer B which has a speed of 1000, a memory of size 64, a hard disk of size 80 and a cost of 550. You will be given int[]s speed, memory, disk, and price. Corresponding elements of the four int[]s represent a single computer. You are to find the indices (starting from 0) of all computers for which some other computer in the list is definitely better. You should put all of these indices in a int[] sorted in ascending order, and return that.

  • Each element of disk will be between 1 and 1000, inclusive.
  • Each element of price will be between 100 and 10000, inclusive.
