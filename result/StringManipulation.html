Problem Statement for "StringManipulation"

Problem Statement

You are to manipulate a given String text, which may contain so-called manipulators. Manipulators are the characters '#' , '$' and '@'. Scan the string from left to right, and whenever you encounter a manipulator, consider the substring between it and the previously encountered manipulator (excluding both manipulators). If there is no previous manipulator, use the beginning of the string instead. Perform one of the following operations on that substring:

  • If the manipulator is '#', reverse the substring.
  • If the manipulator is '@', perform a circular rotation of the substring one position to the right.
  • If the manipulator is '$', perform a circular rotation of the substring one position to the left.

All manipulators in the string should be erased after all manipulations are accomplished. You are to return the resulting string.

Definition

Class:
StringManipulation
Method:
manipulate
Parameters:
String
Returns:
String
Method signature:
String manipulate(String text)
(be sure your method is public)

Constraints

  • text will contain between 1 and 50 characters, inclusive.
  • text will contain only lowercase letters ('a'-'z'), '#', '@' and '$' characters.
  • No two manipulators will be adjacent in text.
  • The first character of text will not be '@', '#' or '$'.

    Returns: "w"

    Returns: "rb"

  3. "qh$"
