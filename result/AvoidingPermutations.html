Problem Statement for "AvoidingPermutations"

Problem Statement

A permutation P of length N is called (2,0,1)-avoiding if there are no indices 0 &lt;= i &lt; j &lt; k &lt; N such that P[i] > P[k] > P[j].


Farmer Bob used to remember a (2,0,1)-avoiding permutation P. However, in an unfortunate accident with his combine, he lost most of his memory. Now, he only remembers its length N and first element P[0]=F. He tried to recover it, but quickly realized that there are many possible solutions.


You're given two ints N and F. Compute and return the number of (2,0,1)-avoiding permutations that farmer Bob could've remembered originally, modulo 1,000,000,007.

Definition

Class:
AvoidingPermutations
Method:
countPermutations
Parameters:
int, int
Returns:
int
Method signature:
int countPermutations(int N, int F)
(be sure your method is public)

Notes

  • A permutation of length N is a sequence of numbers 0 through N-1 such that each number appears in the sequence exactly once.

Constraints

  • N will be between 1 and 5,000, inclusive.
  • F will be between 0 and N-1, inclusive.
