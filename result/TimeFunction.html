Problem Statement for "TimeFunction"

Problem Statement

Class Name:        TimeFunction
===========

Method Signature:  public String subtractTime ( String, String );
=================

Given:
======
Two Strings that represent times where time1 (first argument) is greater than
or equal to time2 (second argument). 
The times are expressed as Strings formatted as "hh:mm:ss:fff" where:
  hh = hours (00 - 99)
  mm = minutes (00 - 59)
  ss = seconds (00 - 59)
  fff = milliseconds (000 - 999). 
Return:
=======
The result of time1 - time2 expressed as a String with the same format as the
arguments.

Example:
========
Arg1 = 21:36:41:953
Arg2 = 18:15:13:943

Should return:
03:21:28:010
