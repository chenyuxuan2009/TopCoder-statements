Problem Statement for "SetFunctions"

Problem Statement

You are given 2 int[]s representing sets of integers. In addition, you will be given a String op that determines the operation you will perform on the 2 given sets. op can be one of (quotes for clarity):
  • "UNION" : Return a list of the integers that occur in either of the two sets or both.
  • "SYMDIFF" : Return a list of the integer that occur in either of the two sets but not both.
  • "INTERSECTION" : Return a list of the integers that occur in both of the two sets.
  • "MINUS" : Return a list of the integers that occur in set1 but not in set2.
Before returning the resultant int[] remove all duplicates, and sort the list into increasing order. See examples for further information.

SetFunctions
Method:

Parameters:

Returns:
int[]
Method signature:

(be sure your method is public)

Constraints

  • set1 will contain between 0 and 50 elements inclusive.
  • set2 will contain between 0 and 50 elements inclusive.
  • Each element in set1 will be between -1000 and 1000 inclusive.
  • Each element in set2 will be between -1000 and 1000 inclusive.
  • Each element in set1 will be distinct.
  • Each element in set2 will be distinct.
  • op will be one of (quotes for clarity): "UNION", "SYMDIFF", "INTERSECTION", or "MINUS"
