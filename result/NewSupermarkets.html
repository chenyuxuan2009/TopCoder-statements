Problem Statement for "NewSupermarkets"

Problem Statement

There are some ranches along a highway. You are given a int[] position, where position[i] is equal to the distance from the start of the highway to the i-th ranch. You are also given a int[] maxDistance, where maxDistance[i] is the maximum distance that the owners of the i-th ranch are willing to travel to get to the nearest supermarket.

Return the minimal number of supermarkets that must be constructed along the highway such that all the ranch owners will be customers of some supermarket.

Definition

Class:
NewSupermarkets
Method:
minNumber
Parameters:
int[], int[]
Returns:
int
Method signature:
int minNumber(int[] position, int[] maxDistance)
(be sure your method is public)

Notes

  • Two or more ranches can have equal positions.

Constraints

  • position will contain between 1 and 50 elements, inclusive.
  • Each element of position will be between 0 and 1000, inclusive.
  • maxDistance will contain exactly the same number of elements as position.
  • Each element of maxDistance will be between 0 and 1000, inclusive.

    {6, 7}

    Returns: 1

    We need to build a supermarket near each ranch.
