Problem Statement for "CuttingPaper"

Problem Statement

You have a piece of paper that is divided into multiple sections with different colors, and you want to cut it into several pieces that each contain only a single color. You want the resulting pieces to be as big as possible, so you will never cut through a single-colored section. You will only make cuts along lines that separate sections with different colors.

A single cut is done in the following way: First, position the cutter on some point on the paper that is not completely surrounded by paper (in other words, a point on an edge). Then, perform a continuous cut until you touch some other point that is not completely surrounded by paper. This entire action counts as a single cut. Note that as soon as a point becomes part of a cut, it is no longer considered to be a point completely surrounded by paper. Therefore, a cut can never cross itself (see example 5). Also note that a cut does not need to be a straight line.

You will be given the paper as a String[], where the jth character of the ith element represents the square at row i, column j. A space character means that there is no paper in that square, and each uppercase letter represents a different color. Return the minimum number of cuts (as described above) needed to accomplish this task. If it is impossible, return -1. See examples for further clarification.

Notes

  • Two squares of the paper connected by only one vertex are not considered connected. This means you do not need to cut the shared corner. See example 1 for further clarification.
  • The paper can contain holes and several separated pieces can be described in the input (see example 1 for clarification).
  • If the ith element of paper contains less than j+1 characters, it means that in row i, there is no paper at any column greater than or equal to j. See example 1 for further clarification.

Constraints

  • paper will contain between 1 and 50 elements, inclusive.
  • Each element of paper will contain between 1 and 50 characters, inclusive.
  • Each character of each element of paper will be a space (' ') or an uppercase letter ('A'-'Z').
