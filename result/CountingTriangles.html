Problem Statement for "CountingTriangles"

Problem Statement

We have a single equilateral triangle. We will now perform a sequence of n steps. In each step, we will take each of the currently smallest equilateral triangles and we will subdivide it into four identical triangles, each half the size of the original one.

You are given the long n. Let X be the number of different ways in which one can find an equilateral triangle in the picture obtained after n steps. As X can be very large, compute and return the value (X modulo 1,000,000,007).

The first two steps are shown in the figure below.
