Problem Statement for "NiceMultiples"

Problem Statement

Consider all multiples of a positive integer M that do not exceed U. If we write each of them in base B, how many of them don't contain any zeros? Return the answer modulo 10^9 + 7.

Definition

Class:
NiceMultiples
Method:
count
Parameters:
long, long, int
Returns:
int
Method signature:
int count(long M, long U, int B)
(be sure your method is public)

Constraints

  • M will be between 1 and 10^12, inclusive.
  • U will be between M and 10^12, inclusive.
  • B will be between 2 and 16, inclusive.

Examples

    The multiples of 1 that do not contain any zeros in base 2 are the numbers 1, 3, 7, 15, 31, and 63. (In base 2, these are 1, 11, 111, 1111, 11111, and 111111.)
