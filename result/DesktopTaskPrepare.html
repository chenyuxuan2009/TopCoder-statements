Problem Statement for "DesktopTaskPrepare"

Problem Statement

Your desktop contains several icons. Each time you press an arrow key, the focus may switch from one icon to another. You are given a String[] icons, each element of which represents a single icon on the desktop. Each element should be formatted as "Name NameUp NameDown NameLeft NameRight" (quotes for clarity), where Name is the name of an icon. When that icon has the focus and you press up, down, left, or right, focus will switch to the icon with name NameUp, NameDown, NameLeft, or NameRight, respectively. Note that the focus may not always change, so one or more of those four names might be equal to Name itself. In this problem, you are asked to check that the input satisfies the following constraints:
  • Constraint 0: Each element of icons will be formatted as described in problem statement;
    For each icon, Name, NameUp, NameDown, NameLeft, NameRight will contain between 1 and 5 uppercase letters ('A' - 'Z').
  • Constraint 1: All names of icons will be different.
  • Constraint 2: Each of NameUp, NameDown, NameLeft, NameRight will be equal to the name of some icon.


Return the String "VALID" if icons satisfies all of the above constraints. Otherwise, return "INVALID: Constraint X violated", where X is the number of the lowest violated constraint ('0', '1', or '2'). This means you must first check Constraint 0 on every element of icons. If all the elements satisfy that constraint, check Constraint 1, and so on. All quotes are for clarity.

Constraints

  • icons will contain between 1 and 50 elements, inclusive.
  • Each element of icons will contain between 1 and 50 characters, inclusive.
  • Each character in icons will have ASCII code between 32 and 126, inclusive.
