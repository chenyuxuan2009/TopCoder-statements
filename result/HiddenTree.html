Problem Statement for "HiddenTree"

Problem Statement

In this problem a directed forest is an arbitrary directed acyclic graph in which each node has at most one incoming edge.

Suppose we have a directed forest and we have already assigned a positive integer weight a[i] to each node i of the forest. We can now compute aggregate weights b[i] as follows: b[i] will be the sum of the weights of all nodes reachable from i, including node i itself.

You are given the int[] b. We say that a directed forest is compatible with b if there is at least one way to assign a positive integer weight a[i] to each node i of the forest in such a way that the aggregate weights of all vertices will be exactly equal to the values given in b.

Compute and return the number of directed forests that are compatible with the given b, modulo 10^9 + 7. Note that each valid forest should only be counted once, even if there are multiple valid ways to assign weights to its vertices.

Definition

Class:
HiddenTree
Method:
count
Parameters:
int[]
Returns:
int
Method signature:
int count(int[] b)
(be sure your method is public)

Constraints

  • b will be contain between 1 and 14 elements, inclusive.
  • Each element of b will be between 1 and 10^9, inclusive.
