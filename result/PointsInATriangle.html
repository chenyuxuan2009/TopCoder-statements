Problem Statement for "PointsInATriangle"

Problem Statement

Given a triangle on a plane, you must find the number of points with integer coordinates which are inside the given triangle. The points on the boundary must not be counted.

You will be given int[]s x and y, where (x[i], y[i]) represents the i-th vertex of the triangle.

Definition

Class:
PointsInATriangle
Method:
countPoints
Parameters:
int[], int[]
Returns:
int
Method signature:
int countPoints(int[] x, int[] y)
(be sure your method is public)

Constraints

  • x and y will each contain exactly three elements.
  • Each element of x will be between -1000 and 1000, inclusive.
  • Each element of y will be between -1000 and 1000, inclusive.
  • The triangle specified by x and y will have non-zero area.
