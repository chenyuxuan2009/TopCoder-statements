Problem Statement for "Meteor"

Problem Statement

You are a sorceress trying to kill cows on the secret cow level. The level is represented as a square with x/y coordinates from 0 to 100. The cows are located at some points on this square with integer coordinates. The only spell available to you is Meteor. You can center Meteor at any point with integer coordinates and Meteor will half-kill all the cows within a given distance from its center.

Your task is, given the locations of the cows and the radius within which the Meteor produces damage, return the minimum number of times you need to fire the Meteor to kill all the cows.

Definition

Class:
Meteor
Method:
numHits
Parameters:
int[], int[], int
Returns:
int
Method signature:
int numHits(int[] xCoor, int[] yCoor, int radius)
(be sure your method is public)

Notes

  • xCoor represents x coordinates of the cows. yCoor represents y coordinates of the cows in the same order.
  • The cows on the border of the Meteor get damage. For example, if there is a cow at coordinates 1,1 and the Meteor is fired with the radius 1 at the point 1,2, the cow gets damage.
  • You need to hit each cow twice (not necessarily firing Meteor at the same point) to kill it.

Constraints

  • xCoor has between 1 and 18 elements inclusive
  • yCoor has the same number of elements as xCoor
  • each element of xCoor and yCoor is between 0 and 100 inclusive
  • radius is between 1 and 100 inclusive
