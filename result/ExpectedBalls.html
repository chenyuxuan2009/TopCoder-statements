Problem Statement for "ExpectedBalls"

Problem Statement

You have N balls and N buckets. Independently, and uniformly at random, you place the balls in the buckets. Given 1 <= N <= 10, return the expected number of balls that will end up in the bucket with the most balls out of any bucket.

Definition

Class:
ExpectedBalls
Method:
expected
Parameters:
int
Returns:
double
Method signature:
double expected(int N)
(be sure your method is public)

Constraints

  • N will be between 1 and 10, inclusive.

Examples

  1. 1

    Returns: 1.0

    The one ball always ends up in the one bucket.

  2. 2

    Returns: 1.5

    Half the time the two balls end up in the same bucket, half the time they end up in different buckets.
