Problem Statement for "DancingLessons"

Problem Statement

To prepare for a dancing contest, you must attend k lessons.

There is one dancing lesson per day available for the following n days. To go to the lesson on the i-th day you must attend at least a[i] of the previous lessons. Attending the lesson on the i-th day costs cost[i].

Return the minimal amount you must pay to prepare for the contest. If it is impossible to prepare for the contest, return -1.

Definition

Class:
DancingLessons
Method:
minCost
Parameters:
int[], int[], int
Returns:
int
Method signature:
int minCost(int[] a, int[] cost, int k)
(be sure your method is public)

Constraints

  • a will contain between 1 and 50 elements, inclusive.
  • Each element of a will be between 0 and 50, inclusive.
  • cost will contain the same number of elements as a.
  • Each element of cost will be between 1 and 1000, inclusive.
  • k will be between 1 and 50, inclusive.
