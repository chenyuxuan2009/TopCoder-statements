Problem Statement for "SingleServing"

Problem Statement

You would like to make a single serving of a recipe designed to yield several servings. You are given the recipe as a String[], recipe, where each element is formatted as "AMOUNT NAME" (quotes for clarity only). NAME is the name of the ingredient, and AMOUNT is an integer representing the number of units required of that ingredient. You are given an int, servings, that represents the number of servings that the recipe yields. Determine how much of each ingredient is required to make a single serving. Return a int[], where each the ith element represents the minimum number of whole units required of ingredient i (the ith element of recipe).

Definition

Class:
SingleServing
Method:
ingredients
Parameters:
String[], int
Returns:
int[]
Method signature:
int[] ingredients(String[] recipe, int servings)
(be sure your method is public)

Constraints

  • recipe contains between 1 and 10 elements, inclusive.
  • Each element of recipe is formatted as "AMOUNT NAME" (quotes for clarity only).
  • Each AMOUNT is an integer between 1 and 20, inclusive, with no leading zeros.
  • Each NAME contains between 1 and 10 uppercase letters ('A'-'Z'), inclusive.
  • servings is between 2 and 50, inclusive.
