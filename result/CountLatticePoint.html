Problem Statement for "CountLatticePoint"

Problem Statement

Consider a two dimensional plane.

Given an int N, return the number of lattice points whose Manhattan distance from (0, 0) is strictly less than N (including the point (0, 0).)

Here's the example for small cases. The center point in each case is (0, 0).
