Problem Statement for "PointSeq"

Problem Statement

Given a sequence of points P and an empty result list L, you will perform the following algorithm:
  • 1) Sort P into ascending order by x-coordinate
  • 2) Remove the first point of P and add it to the result list L
  • 3) Remove the first point of P and add it to the end of result list L
  • 4) Remove the first point of P and add it to the end of result list L
  • 5) While L has 3 or more points, and the last three points of L do not make a right turn, remove the second to last element of L
  • 6) If P is not empty goto step 4.
3 points form a right turn if the third point falls below a line through the first two points. To simplify calculations, all of the points given will have different x-coordinates.

P will be given as two int[]s xs and ys. The kth point will have x and y coordinates determined by the kth elements of xs and ys respectively. Your method will return L as a String[] with each element formatted as "x-coord,y-coord". x-coord and y-coord should have no extra leading zeros. The elements of L should be returned in exactly the order produced by the algorithm above.

Constraints

  • xs will contain between 3 and 50 elements inclusive
  • xs will contain the same number of elements as ys
  • Each element of xs will be between -100000 and 100000 inclusive
  • Each element of ys will be between -100000 and 100000 inclusive
  • xs will contain no repeated elements
